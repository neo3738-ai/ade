# ADE — Automated Development Environment

AI-assisted development workflow manager with task tracking, watchdog monitoring, and MCP integration.

## Features

- **Task management** — create and track development tasks with status, priority, and time estimates
- **Watchdog** — monitors running tasks and alerts when they exceed expected time
- **MCP server** (`ade_mcp`) — AI assistant integration for reading and updating tasks
- **Desktop UI** (`ade_ui`) — full interface for managing your dev workflow

## Binaries

ADE ships two executables:
- `ade_ui` — the desktop application
- `ade_mcp` — the MCP server (add to your AI assistant's MCP config)
